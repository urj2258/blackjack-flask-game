# Blackjack Game (Python)

Welcome to my **Blackjack Game** built during the 100 Days of Code challenge.

## Game Description
- Classic console Blackjack game.
- Player vs Dealer with standard rules.
- Dealer stands on 17, player can hit or stand.
- Includes score calculation and bust logic.

## How to Run
1. Make sure Python 3.x is installed.
2. Open terminal in this folder.
3. Run the game with:  
```bash
python main.py
